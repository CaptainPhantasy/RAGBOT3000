## Step-by-Step Workflow (2025)
1. Click **Gear icon**
2. Select **All Lists** under Lists
3. Choose desired list:
   - **Products and Services**
   - **Customers**
   - **Suppliers**
   - **Chart of Accounts**
   - **Payment Methods**
   - **Terms**
   - **Classes** (if enabled)
   - **Locations** (if enabled)
4. Use **New** button to add items
5. Use **Batch actions** for multiple items
6. Export lists to Excel if needed

## Why This Matters (Accounting Context)
Lists provide:
- Master data for transactions
- Consistency in entries
- Quick access to information
- Bulk management capabilities

## Common Error States / Pitfalls

### Inactive items still in use
- **What you see:** Can't delete items
- **Diagnosis:** Items have transaction history
- **Fix:**
  1. Make item inactive instead of deleting
  2. Merge duplicates if appropriate

### Too many similar items
- **What you see:** Multiple versions of same item
- **Diagnosis:** Created new instead of using existing
- **Fix:**
  1. Merge duplicate items
  2. Establish naming conventions
