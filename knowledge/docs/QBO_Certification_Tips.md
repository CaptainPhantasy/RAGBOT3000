# QuickBooks Online Certification Exam Tips

## Exam Overview
The QuickBooks Online Certification tests your ability to navigate QBO, understand accounting principles, and help clients manage their books effectively.

## Key Topics to Master

### 1. Bank Feed Management
- **Critical**: Understanding Add vs Match
- Add creates NEW transaction
- Match links to EXISTING transaction
- Always use Match for customer payments to invoices

### 2. Undeposited Funds
- **Must Know**: Why use Undeposited Funds
- Groups multiple payments into single deposit
- Matches how bank actually shows deposits
- Skipping creates reconciliation nightmares

### 3. Sales Tax
- Setting up tax agencies
- Configuring taxable items
- Customer tax exemptions
- Filing and payment workflows

### 4. User Permissions
- Different user types (Standard, Admin, Reports Only, Time Tracking Only)
- When to use each type
- Security best practices
- Segregation of duties

### 5. Import Process
- Using templates correctly
- Data mapping
- Common import errors
- Duplicate prevention

## Common Exam Questions

### Question Type: "What should you do when..."

**Scenario**: Customer payment appears in bank feed, and there's an outstanding invoice.
- **Wrong**: Add the transaction
- **Right**: Match to the invoice

**Scenario**: Multiple checks received, deposited together at bank.
- **Wrong**: Record each payment directly to bank account
- **Right**: Receive payments to Undeposited Funds, then create single Bank Deposit

### Question Type: "Which report shows..."

- **Outstanding customer balances**: A/R Aging Summary
- **Outstanding vendor bills**: A/P Aging Summary
- **Profit for a period**: Profit and Loss (P&L)
- **What company owns/owes**: Balance Sheet
- **All transactions in an account**: General Ledger

### Question Type: "Where do you find..."

- **Add new customer**: + New → Customer OR Sales → Customers → New
- **Import data**: Gear icon → Import Data
- **Recurring transactions**: Gear icon → Recurring Transactions
- **User management**: Gear icon → Manage Users

## Study Strategies

1. **Practice in QBO**
   - Create test company
   - Run through all workflows
   - Try different transaction types

2. **Focus on WHY**
   - Understand accounting principles
   - Know why specific workflows exist
   - Understand impact on financials

3. **Navigation Mastery**
   - Know where everything is
   - Practice using + New menu
   - Understand Gear menu organization

4. **Error Scenarios**
   - Know common mistakes
   - Understand how to fix them
   - Prevention strategies

## Exam Day Tips

- Read questions carefully
- Watch for "best" vs "correct" - sometimes multiple answers seem right
- Think about accounting impact
- Consider the QuickBooks workflow
- When in doubt, think about what creates least cleanup

## Key Terms to Know

- **Accrual Accounting**: Record when earned/incurred (not when paid)
- **Cash Basis Accounting**: Record when cash changes hands
- **Accounts Receivable**: Money customers owe you
- **Accounts Payable**: Money you owe vendors
- **Undeposited Funds**: Holding account for payments before deposit
- **Bank Reconciliation**: Matching QBO to bank statement
- **Chart of Accounts**: List of all account categories
- **Bank Feed**: Automatic download of bank transactions
