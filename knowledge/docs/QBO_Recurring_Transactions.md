# QuickBooks Recurring Transactions

## Visual UI Anchors
- **Gear icon** â†’ **Recurring Transactions**
- **New template button**: Create recurring template
- **Type dropdown**: Invoice, Bill, Expense, etc.
- **Schedule options**: Daily, weekly, monthly, custom

## Step-by-Step Workflow (2025)
1. Click **Gear icon**
2. Select **Recurring Transactions** under Lists
3. Click **New**
4. Select **Transaction type** (Invoice, Bill, etc.)
5. Fill out transaction template
6. **Set schedule**:
   - **Type**: Scheduled, Unscheduled, Reminder
   - **Frequency**: Daily, Weekly, Monthly, Yearly
   - **Start date**: When to begin
   - **End**: Never or specific date
   - **Days in advance**: When to create
7. Choose what to do with reminders:
   - **Add to my reminders**: Manual review
   - **Automatically**: Create without review
8. Click **Save template**

## Why This Matters (Accounting Context)
Recurring transactions:
- Automate regular entries (rent, subscriptions)
- Ensure consistent billing (retainer clients)
- Reduce forgotten invoices
- Save time on repetitive entries

## Common Error States / Pitfalls

### Template not running
- **What you see:** Expected transaction not created
- **Diagnosis:** Schedule not set correctly
- **Fix:**
  1. Check start date and frequency
  2. Verify template is "Scheduled" type
  3. Check days in advance setting

### Wrong amounts
- **What you see:** Recurring transaction has outdated pricing
- **Diagnosis:** Template not updated after price changes
- **Fix:**
  1. Open recurring transaction template
  2. Update amounts
  3. Save changes

### Duplicate recurring entries
- **What you see:** Same transaction created multiple times
- **Diagnosis:** Multiple templates for same transaction
- **Fix:**
  1. Review all recurring transactions
  2. Delete duplicates
  3. Keep only one per recurring need
