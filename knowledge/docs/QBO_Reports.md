# QuickBooks Reports

## Visual UI Anchors
- **Reports** (left navigation)
- **Standard, Custom, Management** tabs
- **Search bar**: Find specific reports
- **Customize button**: Modify report columns/filters
- **Run report button**: Generate report
- **Export icons**: Excel, PDF, print options

## Step-by-Step Workflow (2025)
1. Click **Reports** in left navigation
2. Browse or search for report:
   - **Standard tab**: All built-in reports
   - **Search**: Type report name
3. Select the report
4. Customize if needed:
   - Click **Customize**
   - Adjust **Rows/Columns**
   - Set **Filters** (date range, customers, etc.)
   - Click **Run report**
5. **Save customization** for future use
6. **Export** or **Print** as needed

## Key Reports to Know
- **Profit & Loss**: Income minus expenses for a period
- **Balance Sheet**: Assets, Liabilities, Equity as of a date
- **Cash Flow**: Cash in/out from operations, investing, financing
- **A/R Aging**: Outstanding customer balances
- **A/P Aging**: Outstanding vendor bills
- **General Ledger**: All transactions by account

## Common Error States / Pitfalls

### Wrong date range
- **What you see:** Incorrect totals or trends
- **Diagnosis:** Report period not set correctly
- **Fix:**
  1. Click Customize
  2. Set correct date range
  3. Re-run report

### Missing transactions
- **What you see:** Report doesn't include recent activity
- **Diagnosis:** Date range or filter issue
- **Fix:**
  1. Check report date range
  2. Clear filters
  3. Re-run report

### Not understanding report differences
- **What you see:** Confused between P&L and Balance Sheet
- **Diagnosis:** Report purpose unclear
- **Fix:**
  - **P&L**: Shows activity for a period
  - **Balance Sheet**: Shows position at a point in time
  - P&L net income flows to Balance Sheet equity
