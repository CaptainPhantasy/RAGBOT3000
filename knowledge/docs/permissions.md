# Roles & Permissions

- **Owner**: Full access, billing management.
- **Admin**: Can manage members and integrations, cannot delete workspace.
- **Member**: Can create and edit pages.
- **Guest**: Read-only access to specific shared pages.

To change a role: Go to Settings > Members, find the user, and select the dropdown menu.
