# QuickBooks Online Interface Navigation

## Visual UI Anchors
- **Dashboard**: Default page with overview widgets
- **Left Navigation Bar**: Main menu with Banking, Sales, Expenses, etc.
- **+ New button**: Top of navigation bar, green plus sign
- **Gear icon**: Top-right corner, shaped like a gear/cog
- **Search/Magnifying glass**: Top center of the interface
- **Privacy toggle**: Top of Dashboard (eye icon)

## Step-by-Step Workflow (2025)

### Dashboard Navigation
- Home dashboard shows: checklists, bank account details, P&L, expenses, recent invoices
- Cash flow dashboard shows: Money in vs Money out calculations
- Toggle **Privacy mode** to hide sensitive financial data

### Using the Navigation Bar
- **Banking**: Connect bank accounts, review transactions, reconcile
- **Sales**: Manage invoices, estimates, customers, products & services
- **Expenses**: Track bills, expenses, suppliers
- **Reports**: Access all financial reports
- **Taxes**: Manage sales tax and payroll tax
- **Mileage**: Track business mileage
- **Projects**: Track job profitability (if enabled)

### + New Menu
- Four categories: **Customers**, **Suppliers**, **Employees**, **Other**
- Primary access point for creating all transactions
- Includes: Invoice, Estimate, Sales receipt, Bill, Expense, Journal entry, etc.

### Gear Menu Categories
- **Your Company**: Account and Settings, Manage users, Custom form styles
- **Lists**: All Lists, Products and services, Recurring transactions
- **Tools**: Import data, Reconcile, Budgeting, Audit log
- **Profile**: Account details, Billing, Subscription

## Why This Matters (Accounting Context)
Understanding the navigation helps you work efficiently and find features when you need them. Each section is organized by business function, mirroring how accounting workflows operate in practice.

## Common Error States / Pitfalls

### Getting lost in the interface
- **What you see:** Can't find the right screen or feature
- **Diagnosis:** Using the wrong navigation path
- **Fix:** Use the **Search** feature (magnifying glass) to find transactions or features quickly

### Accidentally switching to Business view
- **What you see:** Different layout than expected
- **Diagnosis:** Settings may have switched views
- **Fix:** Gear icon → Account and Settings → Advanced → Switch to Accountant view if preferred
