# QuickBooks Expenses and Receipts

## Visual UI Anchors
- **+ New** → **Expense**
- **+ New** → **Check**
- **Banking** → **Receipts** tab
- **Expenses** → **Expenses** tab
- **Mobile app** receipt capture
- **Gear icon** → **Expense settings**

## Step-by-Step Workflow (2025)

### Entering Expenses
1. Click **+ New** → **Expense**
2. Select **Payee** (who was paid)
3. Select **Payment account** (how paid)
4. Enter **Payment date**
5. Add **expense details**:
   - Select **Account category**
   - Enter **Description**
   - Enter **Amount**
   - Add **Receipt** if available
6. Mark as **Billable** if customer will reimburse
7. Select **Customer** if billable
8. Click **Save and close**

### Managing Receipts
1. **Via mobile app**:
   - Open QuickBooks app
   - Tap Receipts
   - Take photo of receipt
   - Review and save
2. **Via desktop**:
   - Go to **Banking** → **Receipts**
   - Drag/drop or browse for receipt files
   - Match to existing expense or create new
3. Review and categorize matched receipts

## Why This Matters (Accounting Context)
Proper expense tracking ensures:
- Complete tax deductions
- Accurate profit calculation
- Business expense separation from personal
- Project cost tracking (if billable)

## Common Error States / Pitfalls

### Personal expenses coded as business
- **What you see:** Personal meals, groceries as business expenses
- **Diagnosis:** Poor understanding of deductible expenses
- **Fix:**
  1. Review expenses with accountant
  2. Move to Owner's Draw account if needed

### Missing receipts
- **What you see:** Expenses without supporting documentation
- **Diagnosis:** Receipts not retained
- **Fix:**
  1. Use mobile app for immediate capture
  2. Request email receipts when possible

### Duplicate expenses
- **What you see:** Same expense entered twice
- **Diagnosis:** Bank feed added + manual entry
- **Fix:**
  1. Find duplicate
  2. Delete one entry
  3. Use bank feed for most expenses
