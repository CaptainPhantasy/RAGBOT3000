## Step-by-Step Workflow (2025)
1. Click **Gear icon**
2. Select **Import Data** under Tools
3. Select what to import (Customers, Suppliers, etc.)
4. **Download the sample template**
5. Fill out template with your data:
   - Keep header row
   - Don't change column order
   - Use exact names for existing items
6. Save as Excel or CSV file
7. Browse to your file
8. Click **Next**
9. **Map your data** (if not using template)
10. Review import summary
11. Click **Import**

## Why This Matters (Accounting Context)
Importing data:
- Saves manual entry time
- Reduces typing errors
- Maintains existing data structure
- Enables faster QuickBooks setup

## Common Error States / Pitfalls

### Data formatting errors
- **What you see:** Import fails or partial success
- **Diagnosis:** Incorrect date format, missing fields
- **Fix:**
  1. Use the provided template exactly
  2. Check date formats (MM/DD/YYYY)
  3. Ensure required fields are filled

### Duplicate data
- **What you see:** Same customer appears multiple times
- **Diagnosis:** Import without checking existing data
- **Fix:**
  1. Export existing data first
  2. Remove duplicates from import file
  3. Or use Add/Update option if available

### Incorrect mapping
- **What you see:** Data in wrong fields
- **Diagnosis:** Column mapping incorrect
- **Fix:**
  1. Carefully review mapping before import
  2. Test with small batch first
