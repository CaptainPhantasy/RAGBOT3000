# QuickBooks Settings and Customization

## Visual UI Anchors
- **Gear icon** → **Account and Settings**
- **Categories**: Company, Payments, Expenses, etc.
- **Pencil icon**: Edit sections
- **Save button**: Apply changes
- **Advanced tab**: Additional options

## Step-by-Step Workflow (2025)
1. Click **Gear icon**
2. Select **Account and Settings** under Your Company
3. Navigate tabs:
   - **Company**: Name, address, email, industry
   - **Payments**: Accept payments online
   - **Sales**: Sales form content, messages
   - **Expenses**: Bills, expenses, purchase orders
   - **Advanced**: Accounting, categories, automation
4. Click **Pencil** to edit section
5. Make changes
6. Click **Save**

## Key Settings to Review
- **Company information**: Legal name, address
- **Sales forms**: Custom fields, default messages
- **Sales tax**: Tax agency setup
- **Terminology**: Customer vs Client, etc.
- **Automation**: Prefill forms, default accounts
- **Time tracking**: Single vs weekly time

## Why This Matters (Accounting Context)
Proper settings ensure:
- Accurate default information

- Proper tracking capabilities

## Common Error States / Pitfalls

### Wrong accounting method
- **What you see:** Cash vs Accrual issues
- **Diagnosis:** Incorrect selection in Advanced settings
- **Fix:**
  1. Go to Advanced → Accounting
  2. Select correct method
  3. Consult accountant first

### Duplicate warnings disabled
- **What you see:** Duplicate invoice/check numbers
- **Diagnosis:** Warning features turned off
- **Fix:**
  1. Go to Advanced → Other preferences
  2. Enable duplicate warnings
