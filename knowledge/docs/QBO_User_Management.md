## Step-by-Step Workflow (2025)
1. Click **Gear icon**
2. Select **Manage Users** under Your Company
3. Click **Add user**
4. Select **User type**:
   - **Standard user**: Access to selected areas
   - **Company admin**: Full access
   - **Reports only**: View reports only
   - **Time Tracking only**: Enter time only
   - **Custom access**: Select specific permissions
5. Enter user's **email address**
6. Select user's **role** or customize permissions
7. Click **Next**
8. Review permissions
9. Click **Add**
10. User receives email invitation to set up account

## Why This Matters (Accounting Context)
User management ensures:
- Proper segregation of duties
- Data security and privacy
- Appropriate access levels
- Internal controls

## Common Error States / Pitfalls

### Too many admin users
- **What you see:** Multiple users with full access
- **Risk:** Errors harder to track
- **Fix:**
  1. Review user permissions quarterly
  2. Limit admin access to essential users

### Insufficient permissions
- **What you see:** User can't perform needed tasks
- **Diagnosis:** Access rights too restrictive
- **Fix:**
  1. Edit user permissions
  2. Add necessary access

### Former employees still have access
- **What you see:** Departed user can still log in
- **Risk:** Security vulnerability
- **Fix:**
  1. Immediately deactivate user
  2. Consider what to do with their transactions
