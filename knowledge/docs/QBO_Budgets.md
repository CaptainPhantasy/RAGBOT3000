## Step-by-Step Workflow (2025)
1. Click **Gear icon**
2. Select **Budgeting** under Tools
3. Click **Add budget**
4. Enter **Budget name**
5. Select **Fiscal year**
6. Choose **Subdivided by** (Month, Quarter, Year)
7. If using tracking, select by Customer, Class, or Location
8. Enter budget amounts:
   - Copy from previous year
   - Enter manually
   - Import from Excel
9. Click **Save**
10. Run **Budget vs Actuals** report to review

## Why This Matters (Accounting Context)
Budgets help:
- Plan future expenses and revenue
- Compare actual performance to goals
- Make informed business decisions
- Identify areas needing attention

## Common Error States / Pitfalls

### Unrealistic numbers
- **What you see:** Large variances every month
- **Diagnosis:** Budget not based on historical data
- **Fix:**
  1. Review past P&L statements
  2. Adjust budget to be realistic
  3. Update quarterly

### Not reviewing actuals
- **What you see:** Budget created but never used
- **Diagnosis:** No monitoring process
- **Fix:**
  1. Run Budget vs Actuals monthly
  2. Review significant variances
  3. Adjust as needed
